# Recruitment excercise - Voting App

Web API for a voting app implemented as a job recruitment task.

## Technologies
* **Backend:** .NET 10
* **Database:** SQLite, Entity Framework Core
* **Validation:** FluentValidation
* **Tests:** NUnit, Moq
* **Documentation:** Swagger

## How to run?
This project uses a new solution format - .slnx. Older IDE may not open it properly. You can run it via command line to avoid issues.

1. Clone this repository: `git clone [url]` (or download as ZIP)
2. Go to the project directory: `cd VotingApp`
3. Run the application: `dotnet run`
4. Open Swagger: `http://localhost:5049/swagger/index.html`
